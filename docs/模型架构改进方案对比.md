# 残差UNet架构改进方案对比（2024-2025最新文献支撑）

---

## 方案一：小波增强残差UNet（Wavelet-Enhanced Residual UNet）⭐最推荐

### 核心思想
在残差UNet的skip connection中引入**离散小波变换（DWT）高频增强分支**。DWT可以将特征无损分解为低频（LL）和高频（LH/HL/HH）子带，让网络显式地学习和恢复高频细节。

### 为什么适合你
- 你的核心问题是**边缘模糊+纹理丢失** = 高频信息丢失
- DWT天生把高频分离出来，让网络可以**针对性地增强高频子带**
- **不改变原有UNet结构**，只在skip connection上加模块，实现简单
- 小波变换本身**无参**，不增加模型参数量

### 具体实现
```
原始残差UNet的skip connection: Encoder_feat ──→ Decoder

改进后:
Encoder_feat ──→ DWT ──┬──→ LL (低频, 直接传递)
                        ├──→ LH ──→ Conv ──┐
                        ├──→ HL ──→ Conv ──┼──→ Concat ──→ IDWT ──→ Decoder
                        └──→ HH ──→ Conv ──┘
                        
                        高频子带先经过轻量卷积增强，再逆变换融合
```

### 文献支撑
| 文献 | 会议/年份 | 核心结论 |
|------|----------|---------|
| **HDW-SR** | arXiv 2025 | "wavelet-based downsampling in place of standard CNN downsampling to achieve multi-scale frequency decomposition" |
| **Diffusion Transformer meets Multi-level Wavelet Spectrum** | **ICCV 2025** | 多级小波谱Transformer，显式建模多尺度频率关系 |
| **RCDM** | **CVPR 2025 Workshop** | "2D DWT to decompose into frequency sub-bands...processed independently and later fused"，REDS4数据集SSIM 0.9175 |
| **MWCNN** | 2018 (经典) | Multi-level wavelet CNN for image restoration，被大量后续工作引用 |

### 复杂度评估
- **参数量增加**: ~5-10%（只有高频子带的轻量卷积层）
- **计算量增加**: ~10-15%（DWT/IDWT几乎无计算开销，主要是额外卷积）
- **实现难度**: ⭐⭐（中等，PyTorch有现成DWT/IDWT：`torchvision`或`pytorch-wavelets`）

## 方案二：双分支频域-空间域残差UNet（Dual-Branch Spatial-Frequency UNet）

### 核心思想
构建**空间域分支** + **频域分支**的双路径UNet。空间域分支保持原有结构负责整体重建，频域分支通过FFT在频谱层面恢复高频细节，最后两个分支融合输出。

### 为什么适合你
- 频域分支直接优化**频谱重建**，对高频恢复更敏感
- 空间域分支保证**整体结构不崩**
- 双分支互补，比单一路径更鲁棒

### 具体实现
```
输入 ──┬──→ 空间域UNet分支 ──┐
      │                      ├──→ Fusion ──→ 输出
      └──→ FFT ──→ 频域UNet分支 ──┘
      
频域分支: 对输入做FFT ──→ 在频域做卷积 ──→ IFFT回到空间域
```

### 文献支撑
| 文献 | 会议/年份 | 核心结论 |
|------|----------|---------|
| **FDI-VSR** | 2025 (PMC) | Frequency-Spatial Integration Module (FSIM)，2D FFT处理高频细节 |
| **PFESA** | **MICCAI 2025** | "FFT-based Parameter-Free Edge and Structure Attention"，无参数FFT边缘注意力 |
| **GFUNet** | 2023 | Global-Frequency-Domain Network，傅里叶域UNet分割 |
| **FENet** | 2025 | "Multiscale Frequency Domain Enhancement (MSFDE) utilizes Fourier Transform to separate frequency components" |

### 复杂度评估
- **参数量增加**: ~50-80%（几乎是两个UNet）
- **计算量增加**: ~60-100%
- **实现难度**: ⭐⭐⭐（较高，需要设计融合策略）

## 方案三：边缘感知残差块（Edge-Aware Residual Block）

### 核心思想
不改动UNet整体架构，只把普通的残差块（ResBlock）替换为**边缘感知残差块**。每个残差块内部加入Sobel边缘检测分支，对边缘区域的特征进行自适应加权增强。

### 为什么适合你
- **改动最小**，只替换ResBlock，UNet骨架完全不动
- 直接针对**边缘模糊**问题
- 参数量增加极少

### 具体实现
```
原始残差块:
Input ──→ Conv ──→ ReLU ──→ Conv ──→ + ──→ Output
                              ↑______|

边缘感知残差块:
Input ──┬──→ Conv ──→ ReLU ──→ Conv ──→ + ──→ Output
        │                              ↑______|
        └──→ Sobel ──→ Edge_Mask ──→ 加权系数 ──┘
        
Edge_Mask控制残差路径的强度：边缘区域增强，平滑区域抑制
```

### 文献支撑
| 文献 | 年份 | 核心结论 |
|------|------|---------|
| **EatSRGAN** | 2025 | "Edge-Aware Normalized Attention"，显式聚焦高梯度区域，mitigating over-smoothing |
| **FDSE** | 2023 | Four-Direction Sobel Enhancement，四方向Sobel比标准Sobel更全面 |
| **EAGLE** | 2021 (SPIE JMI) | 边缘感知梯度定位损失，比TV/SSIM更好的边缘保持 |
| **PFESA** | MICCAI 2025 | "FFT decoupling for edge (high-freq) vs structure (low-freq)" |

### 复杂度评估
- **参数量增加**: ~2-5%（只有Sobel滤波器+一个1x1卷积）
- **计算量增加**: ~5%
- **实现难度**: ⭐（简单，最容易实现）

## 我的最终建议

**选方案一（小波增强残差UNet），理由：**
1. **ICCV 2025 + CVPR 2025 Workshop** 双顶会支撑，评审认可度高
2. **完美匹配你的缺陷**：DWT把高频（边缘+纹理）显式分离出来，针对性增强
3. **实现难度适中**：不改UNet骨架，只加skip connection模块
4. **参数量增加少**：小波变换无参数，只增加几个轻量卷积层
5. **可以进一步叠加**：如果方案一效果不错，还可以在残差块里再加方案三的边缘感知，形成组合创新

**论文故事线：**
> "现有残差UNet通过重复下采样丢失高频信息，导致边缘模糊和纹理丢失。本文提出在skip connection中引入小波高频增强模块，将编码器特征分解为低频/高频子带，对高频子带进行针对性增强后再融合。实验表明..."
