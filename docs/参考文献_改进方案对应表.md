# 改进方案参考文献列表（2022-2026最新）

## 一、FFT/频域损失类（解决纹理丢失 + 高频恢复）

### 1.1 DiMoSR: Feature Modulation via Multi-Branch Dilated Convolutions for Efficient Image Super-Resolution (2025)
- **核心方法**：MAE + FFT Loss（权重0.05），明确验证了FFT Loss对高频细节恢复的有效性
- **与你的关联**：可直接借鉴"MAE + FFT Loss"的联合训练策略，权重设置有参考值
- **URL**：https://arxiv.org/pdf/2505.21262
- **关键结论**："adding FFT loss helps recover more accurate high-frequency details and sharper edges"

### 1.2 Ni et al., "Misalignment-robust frequency distribution loss for image transformation", CVPR 2024
- **核心方法**：Frequency Distribution Loss (FDL)，在频域计算分布距离，对轻微错位鲁棒
- **与你的关联**：如果你的模型有帧间对齐问题，FDL比直接FFT L1更鲁棒；有官方PyTorch实现
- **URL**：https://github.com/eezkni/FDL
- **论文**：https://arxiv.org/abs/2402.11891

### 1.3 FDI-VSR: Video Super-Resolution Through Frequency Domain Information (2025)
- **核心方法**：Frequency-Spatial Integration Module (FSIM)，用2D RFFT在特征空间做频域处理
- **与你的关联**：视频任务中频域建模的最新工作，说明频域方法在视频恢复中的有效性
- **URL**：https://pmc.ncbi.nlm.nih.gov/articles/PMC12031633/

### 1.4 Cai et al., "Frequency-domain image super-resolution via dual-branch network", 2021/2024
- **核心方法**：DCT频率损失函数，用离散余弦变换做频域监督
- **与你的关联**：DCT比FFT更适合图像块处理，且与HEVC的DCT编码有天然联系
- **URL**：https://arxiv.org/abs/2103.05657（后续有多篇跟进工作）

### 1.5 Flexible Alignment Super-Resolution Network for Multi-Contrast MRI (2022)
- **核心方法**：FFT Loss公式非常简洁：L_FR = ||F(I_SR) - F(I_HR)||_1
- **与你的关联**：最简单的FFT Loss实现参考，直接对图像做FFT后求L1
- **URL**：https://arxiv.org/pdf/2210.03460v1

## 二、边缘保持损失类（解决边缘模糊）

### 2.1 EAGLE: an edge-aware gradient localization enhanced loss for CT image reconstruction (2021, SPIE JMI)
- **核心方法**：Eagle-Loss，边缘感知梯度定位增强损失，缓解模糊和伪影
- **与你的关联**：专门针对"边缘模糊"设计的损失函数，在CT重建中验证了比TV、SSIM更好的边缘保持效果
- **URL**：https://www.spiedigitallibrary.org/journals/journal-of-medical-imaging/volume-12/issue-1/014001/EAGLE--an-edge-aware-gradient-localization-enhanced-loss-for/10.1117/1.JMI.12.1.014001.full
- **关键结论**："Eagle-Loss strikes an optimal balance between noise reduction and detail preservation"

### 2.2 Jiang et al., "Edge loss for image restoration", 2020 (被CMFNet引用)
- **核心方法**：Edge Loss = sqrt(||Δ(X) - Δ(Y)||^2 + ε^2)，Charbonnier形式的边缘损失
- **与你的关联**：最标准的Edge Loss公式之一，ε防止梯度消失，可直接用于你的代码
- **来源论文**：CMFNet (OpenReview), https://openreview.net/pdf/845811e05ad299a2a982761b5961f3cb60aca05c.pdf

### 2.3 EatSRGAN: Edge-Aware Normalized Attention for Efficient and Detail-Preserving Single Image Super-Resolution (2025)
- **核心方法**：Edge-Aware Attention + Edge-guided composite loss，显式聚焦高梯度区域
- **与你的关联**：最新（2025年9月）的边缘保持工作，明确说"mitigating over-smoothing and preserving boundary integrity"
- **URL**：https://arxiv.org/html/2509.14550v1
- **关键结论**："explicitly directing representational capacity toward gradient-dense regions is essential for mitigating over-smoothing"

### 2.4 HOLI-SRNet (2026, Springer)
- **核心方法**：整合pixel-level + perceptual + adversarial + edge-preservation loss的多参数损失框架
- **与你的关联**：边缘保持损失作为多目标优化的一部分，适合资源受限场景
- **URL**：https://link.springer.com/article/10.1186/s13640-026-00689-4

### 2.5 FDSE: Four-Direction Sobel Enhancement Module (2023)
- **核心方法**：四方向Sobel滤波器（水平/垂直/对角线）提取边缘特征，融合到主干网络
- **与你的关联**：Sobel算子的四方向扩展，比标准Sobel更全面地捕捉边缘
- **URL**：https://pmc.ncbi.nlm.nih.gov/articles/PMC12349562/

## 三、过平滑问题理论分析类（支撑根因分析）

### 3.1 Xu et al., "Uncovering the Over-smoothing Challenge in Image Super-Resolution", 2022 (IEEE TPAMI 2024)
- **核心方法**：提出Center-oriented Optimization (COO)问题，从几何和概率角度证明L1/MSE损失导致过平滑
- **与你的关联**：**最核心理论文献**，直接解释为什么你的A模型（L1+SSIM）会过平滑
- **URL**：https://arxiv.org/abs/2201.01034
- **关键结论**："PSNR-oriented models tend to generate over-smoothed images that converge to the center point of similar HRs rather than the ground truth"

### 3.2 Video Dynamics Prior (NeurIPS 2023)
- **核心方法**：分析L1/MSE导致suboptimal quality with edginess in texture，提出spatial pyramid loss + variation loss
- **与你的关联**：视频恢复任务中过平滑问题的分析，验证了L1 alone不够
- **URL**：https://proceedings.neurips.cc/paper_files/paper/2023/file/6ba85c6f1c7656a6a647bc4d63b90bf0-Paper-Conference.pdf

### 3.3 CMFNet (OpenReview)
- **核心方法**：提出PS Loss (PSNR+SSIM组合) + Edge Loss，解决L1/L2的over-smoothing问题
- **与你的关联**：直接对标你的L1_SSIM Loss，给出了"加Edge Loss"的改进路径
- **URL**：https://openreview.net/pdf/845811e05ad299a2a982761b5961f3cb60aca05c.pdf

## 四、视频增强领域最新方法类（整体思路参考）

### 4.1 DAM-VSR: Disentanglement of Appearance and Motion for Video Super-Resolution (2025)
- **核心方法**：dynamic frequency loss，视频超分辨率中动态频率损失
- **与你的关联**：视频任务中的频率损失最新应用
- **URL**：https://arxiv.org/html/2507.01012v1

### 4.2 STAR: Local Information Enhancement + Dynamic Frequency Loss (2025)
- **核心方法**：丰富局部细节 + 缓解退化伪影 + dynamic frequency loss增强保真度
- **与你的关联**："缓解退化伪影"和"增强保真度"正是你的改进方向
- **来源**：被DAM-VSR引用，原文：Xie et al., 2025

### 4.3 Low-Resource VSR using Memory Wavelets and Deformable Convolutions (CVPR 2025 Workshop)
- **核心方法**：wavelet-based feature extraction + memory tensors，避免transformer的过平滑
- **与你的关联**：小波变换作为频域分析的替代方案，避免传统CNN的over-smoothing
- **URL**：https://openaccess.thecvf.com/content/CVPR2025W/WiCV/papers/Viswanathan_Low-Resource_Video_Super-Resolution_using_Memory_Wavelets_and_Deformable_Convolutions_CVPRW_2025_paper.pdf

### 4.4 EvoIR: Towards All-in-One Image Restoration via Evolutionary Frequency Modulation (2025)
- **核心方法**：Evolutionary Frequency Modulation，频率感知调制模块
- **与你的关联**：频率域建模在图像恢复中的最新综述性方法
- **URL**：https://arxiv.org/html/2512.05104v1

## 五、推荐的最小可行改进方案（MVP）

基于以上文献，推荐你**第一步**先做这个最简单的联合损失：

```
L_total = L1 + λ_edge * L_edge + λ_fft * L_fft

其中：
- L1: 原始像素级L1损失（保留，保证PSNR不崩）
- L_edge: ||Sobel(Pred) - Sobel(GT)||_1（参考Jiang 2020 / EatSRGAN 2025）
- L_fft: ||FFT(Pred) - FFT(GT)||_1（参考Flexible Alignment SR 2022 / DiMoSR 2025）
- λ_edge = 0.1 ~ 0.3（参考EatSRGAN的消融实验）
- λ_fft = 0.05（参考DiMoSR的实验设置）
```

**为什么这个方案可行？**
1. **Edge Loss**有EatSRGAN 2025和CMFNet的直接验证
2. **FFT Loss**有DiMoSR 2025和NTIRE 2025多个参赛队的验证
3. **权重设置**有明确文献参考，不需要从零调参
4. **实现简单**：只需要修改`losses/l1_ssim_loss.py`，不需要改模型架构

**下一步验证策略**：
1. 先跑一组ablation（只加Edge Loss / 只加FFT Loss / 两者都加）
2. 对比PSNR/SSIM变化，确认没有崩
3. 用v2分析脚本重新跑一遍，看edge_mae和hf_loss是否下降
